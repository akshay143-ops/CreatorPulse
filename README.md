# CreatorPulse

A command-line analytics program for authors and artists.

## What It Does
CreatorPulse aggregates performance data from multiple platforms and
generates detailed analytics reports using HashMaps as the core data
structure. It tracks views, revenue, follower growth, and engagement
for both authors and artists.
